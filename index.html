Learn About Git & GitHub






Basic Commands Of Gits:-


  • Clone
  • Status
  • Add
  • Commit


Clone Command:-


CLone A Repository On Our Local Machine.



(git clone <-some link->)


How To Use It:
  • Go To Your Github Account.
  • Then Go To Your Repository/folder.
  • Then You see (Code) Button, click On The (Code).
  • After That You See (HTTPS Link), Copy That Link.
  • Then Open Your Visual Studio Terminal.
  • Then Write In The Terminal git clone (paste HTTPS link) Then Press Enter.
  • After All This Process You See Your Created Folder In Your Git Account Into Your Visual Studio. Explorer Bar.



Status Command:-


Displays The Status Of Code.



(git status)


Add Command:-


Add New Or Changed Files In Your Working Directory To The Git Staging Area.



(git add <-file name->)


Commit Command:-


It's The Record Of Change.



(git commit-m "Some Message")


Push Command:-


Upload Local(laptop) Repository/Folder Content to Remote(github) Repository/Folder



(git push Origin Main)


Init Command:-


Used To Create A New Git Repository.



(git init)
